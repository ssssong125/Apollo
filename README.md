주요내용 :
- Thymeleaf로 메인페이지를 구현하였고 네비게이션 바를 사용하여 다른 메뉴로 이동이 가능하도록 구현하였습니다.
- 주요 메뉴로는 상품 판매, 블로그, 질문게시판, FAQ, 장바구니, 결제 등의 메뉴들이 존재하고 있습니다.
- 결제의 경우 카카오 결재 API를 이용하여 테스트 결제가 가능하도록 구현되어 있습니다.
